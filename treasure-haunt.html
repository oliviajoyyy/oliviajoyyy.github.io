Treasure Haunt

This is a game I created using p5.js code for my Art 74 (Digital Media Art) final project.

The objective of the game is to collect certain treasures throughout the house. Use the W, A, S, D keys to control the ghost character and move between rooms by heading toward the green squares. There are 5 levels. As the the levels progress, there are more items to collect and spiders to avoid. After running into a spider three times during a level, you'll have to try again.

I created this game in the way that the ghost can move between rooms because I wanted to give players the freedom to explore.
