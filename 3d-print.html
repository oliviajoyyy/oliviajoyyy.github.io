3D Print

The objective of this project was to export the Minecraft Sculpture as an STL file using a software called Mineways and prepare it for a 3D Print using the printer's software. Using the 3D printer's software, I added scaffolding to the edges so that it had support underneath. I first did small print was as a draft to understand how the sculpture should be rebuilt better if needed. Then I did a final print which was much larger to show the details of the sculpture. My final print of the Iron man helmet took almost 7 hours and came out to be about 3 inches tall. To finish the piece, I removed the scaffolding, filed down the rough edges, and painted it with acrylic paint.
